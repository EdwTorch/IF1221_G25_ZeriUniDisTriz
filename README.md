# Tugas Besar IF1221 Logika Komputasional

## Deskripsi Tubes UNIIII
Proyek ini adalah implementasi dari ~~Tugas Besar~~ Praktikum IF1221: Logika Komputasional 2026.
Proyek ini adalah Implementasi dari Game Bernama Uno yang diparodikan menjadi Uni. 
Permainan ini menggunakan Mayoritas peraturan Utama di dalam Uno Official
Permainan ini melibatkan 2-4 Pemain, 
Pemain dapat memainkan kartu, ambil kartu, menantang, menyebutkan UNI, lihat command, lihat kartu yang dimiliki, dan cek Info  jumlah kartu pemain lain

## Jenis Kartu
1. Kartu Angka
2. Wild Card (+4)
3. Wild Card (Pilih Warna)
4. Draw Two (+2)
5. Skip
6. Reverse

Setiap kartu selain Wild Card, memiliki 4 warna, yaitu merah, kuning, hijau, biru.

## Perintah
1. startGame
2. mainkanKartu(NomorUrutKartuDiTangan)
3. ambilKartu 
4. tantang
5. uni(NomorUrutKartuDiTangan)
6. tangkap(NamaPemain)
7. lihatCommand
8. lihatKartu
9. cekInfo
10. saveGame
11. loadGame
